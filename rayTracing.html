RAY TRACING

Tools: XCode, openFrameworks, C++
Timeline: 12 Weeks [Sept 2020 - Nov 2020]

Ray Tracing is my introduction to Computer Graphics. Over the course of 12 weeks, I learned to render meshes as well as render spheres with ambient lighting, point lighting, Lambert shading, and Phong shading. Additionally, I implemented texture mapping with planes. I later added interactive elements using GUI, allowing the user to create and remove as many spheres and point lights as desired, as well as allowing the user to change the color and size of the spheres and lights.

Next Steps: I would like to successfully implement texture mapping on spheres as well as implenting spotlights.
