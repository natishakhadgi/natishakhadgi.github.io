GLOBAL VIBE CHECK

Team: Jonathan Vo [Project Manager & Full-stack Developer], Annahi Paez [UI Designer], Philip Truong [Frontend Developer],
Role: Frontend Developer
Tools: HTML, CSS, Javascript, React
Timeline: 40 Hours [Jan 2021]

Global Vibe Check is a web application that provides a user-friendly visualization of the globe. This was my first hackathon I participated in and through this experience, I was able to introduce myself to React and APIs.

Next Steps: I would like to improve the use of the Unsplash API when inserting photos to each infographic card so that each populated image is a proper landscape that represents the respective city shown. I would also like to make the web application responsive-friendly.
