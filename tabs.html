TABS | TAYLOR SWIFT ALBUMS

Tools: Visual Studio Code, HTML, CSS, Javascript
Timeline: 1 Week [Jan 2021]

Tabs | Taylor Swift Albums is a small independent project meant to hone in my CSS and Javascript skills. I decided to use Taylor Swift as inspiration since each studio album and era has its own distinct style which would allow for more creativity. Each page reflects an individual album that contains a brief history, tracklist, and accolades.
