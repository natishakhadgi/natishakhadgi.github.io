NURSING MY PREGNANCY

Team: Marysia Tran [Frontend Developer]
Role: Frontend Developer
Tools: HTML, CSS, Javascript, Firebase, React
Timeline: 24 Hours [April 2021]
Award: Best Female Empowerment

Nursing my Pregnancy is a user-friendly virtual platform for volunteer nurses to connect, chat, have video calls with low income, young, and/or single pregnant mothers. Through this hackathon, I became more comfortable creating components in React and connecting to Firebase.

Next Steps: I would like to add additional styling for the chat/video room page. I would also like to add functionality to the Registration page so that I can hone in creating users with Firebase.
