HOME-USER-BACKPACK

Team: Vyshnavi Namani [Frontend Developer], Zeyna Abdulla [Frontend Developer], Amy Li [UX Designer]
Role: Frontend Developer
Tools: HTML, CSS, Javascript, Firebase
Timeline: 12 Hours [February 2021]
Award: Best UI/UX

Home-User-Backpack is a hub for college event planners to help virtually connect classmates, student committees, and members of student organizations by crowdsourcing activities on the internet. I helped brainstorm and design the website and learned to use Firebase for the first time.

Currently: I am currently working revisiting this project so that I may apply the new web development skills I have picked up. I am using MongoDB, Node.js, and React to bring this project to fruition.
